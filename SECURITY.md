# Security Policy

## Supported Versions

The following table shows which versions of OmniLogin-Lite are currently supported with security updates:

| Plugin Version | Minecraft Version | Supported          |
| -------------- | ---------------- | ------------------ |
| 0.0.1-ALFA     | 1.20+            | :white_check_mark: |
| < 0.0.1-ALFA   | < 1.20           | :x:                |

## Reporting a Vulnerability

To report a security vulnerability:
- Please contact the maintainer directly via Discord: [Discord Invite](https://discord.gg/UzXWvshGfh)
- Clearly describe the vulnerability and provide steps to reproduce if possible.
- You will receive an initial response within 72 hours.
- If the vulnerability is accepted, you will be updated on the progress and notified when a fix is released.
- If declined, you will receive an explanation.

Thank you for helping keep OmniLogin-Lite secure!
